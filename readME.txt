GYM MANAGEMENT APPLICATION 

College Assignment - Year 1 - Programming Fundamentals

Create a Terminal Based Gym Manangement System using Java and Object Oriented Programming.

IDE: IntelliJ 

Dependencies: Java 11.0.6 SDK

Functionality:
Create, Update, Delete memeber profiles composed of personal information and physical measurements. 

Member and Trainer logins.

Track fitness progress based on given metrics recorded as Assessments.

Basic analytics performed based on Assessment metrics.
